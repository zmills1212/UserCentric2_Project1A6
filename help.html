Help & Support

Find answers to common questions about using EASYTECH below.

How do I find tutorials?

To find tutorials, click on the "Tutorials" link in the main navigation. From there, you can select either the "iPhone Tutorials" or "Android Tutorials" category to browse the available guides. (Currently, these pages are placeholders).

How do I create a tutorial?

Click on the "Create Tutorials" link in the navigation bar. This will take you to a form where you can enter the title, category, and text for your tutorial. Fill out the required fields and click "Submit". (Note: Tutorial submission is simulated and data is not saved).

Do I need an account?

Currently, Browse and creating tutorials does not require an account. An account might be needed for future features. You can access the login or account creation page via the Login link in the header or the Account link in the navigation.

How does the 'Text Size' button work?

The "Text Size: A A" buttons shown on some pages are placeholders for accessibility features that would allow users to increase or decrease the font size on the page. This functionality is not implemented in the current version.
